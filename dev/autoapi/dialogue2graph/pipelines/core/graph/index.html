load_graph()[source]#

Load graph from dictionary representation into NetworkX DiGraph.

Creates a directed graph from the graph_dict, handling node and edge attributes. Also creates node mapping if node IDs need renumbering.

visualise(*args, **kwargs)[source]#

Visualize the graph using matplotlib and networkx.

Creates a visualization of the graph with nodes and edges labeled with utterances. Uses pygraphviz layout if available, falls back to kamada_kawai_layout.
