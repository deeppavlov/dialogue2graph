Dialog2Graph Documentation
===========================================

**Dialog2Graph** project extends the `Chatsky project <https://github.com/deeppavlov/chatsky>`_ by enhancing `chatsky flows` and `scripts` creation 
with the help of Large Language Models (LLMs). This project is based on the dialog2graph approach which helps to transform conversational data into structured, 
graph-based representations.
