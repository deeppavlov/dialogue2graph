dialogue2graph.pipelines.report#

The module provides creating pipeline reports in various formats.

Classes#

PipelineReport

PipelineReport is a model for generating and managing reports for a service pipeline.

Module Contents#

class dialogue2graph.pipelines.report.PipelineReport(/, **data: Any)[source]#

Bases: pydantic.BaseModel

PipelineReport is a model for generating and managing reports for a service pipeline. It provides functionality to store properties, subreports, and export the report in various formats such as JSON, CSV, Markdown, and plain text.
