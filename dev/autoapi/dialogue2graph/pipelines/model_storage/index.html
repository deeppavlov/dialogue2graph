Validates the model attribute based on the model_type. Ensures that: - If model_type is “llm”, the model must be an instance of BaseChatModel. - If model_type is “emb”, the model must be an instance of HuggingFaceEmbeddings. :raises ValueError: If the model does not match the expected type for the given model_type.

class dialogue2graph.pipelines.model_storage.ModelStorage(/, **data: Any)[source]#

Bases: pydantic.BaseModel

ModelStorage is a class for managing the storage of model configurations and instances. It provides functionality to load configurations from a YAML file, add new models to the storage, and save the current storage state back to a YAML file.
