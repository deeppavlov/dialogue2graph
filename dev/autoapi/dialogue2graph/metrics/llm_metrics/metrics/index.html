dialogue2graph.metrics.llm_metrics.metrics.compare_graphs(G1: dialogue2graph.pipelines.core.graph.BaseGraph, G2: dialogue2graph.pipelines.core.graph.BaseGraph, embedder: str = 'BAAI/bge-m3', sim_th: float = 0.93, llm_comparer: str = 'o3-mini', formatter: str = 'gpt-3.5-turbo', device='cuda:0') → dialogue2graph.pipelines.core.schemas.CompareResponse[source]#

Compare two graphs via utterance embeddings similarity. If similarity is lower than sim_th value LLM llm_comparer is used for additional comparison. LLM formatter is used to keep LLM answer in a required format.

Parameters:

  • G1 (dialogue2graph.pipelines.core.graph.BaseGraph) – graphs to compare

  • G2 (dialogue2graph.pipelines.core.graph.BaseGraph) – graphs to compare

  • embedder (str) – name of embedding model on HuggingFace

  • sim_th (float) – similarity threshold to use LLM

  • llm_comparer (str) – name of LLM to compare graphs

  • formatter (str) – name of LLm modle to format llm_comparer output

Returns:

dict with True or False value and a description.

Return type:
