Chatsky LLM-Autoconfiguration Documentation¶

Chatsky LLM-Autoconfiguration project extends the Chatsky project by enhancing chatsky flows and scripts creation with the help of Large Language Models (LLMs). This project is based on the dialogue2graph approach which helps to transform conversational data into structured, graph-based representations.

The Getting started section contains instructions of how to install the project. It also includes basic terms used in the documentation.
